Featured Project

Health Care Navigation

Genre: VR Simulation Developed: August 2024

Health Care Navigation is a Virtual Reality simulation where the user is given a task to set up a doctor's office in preparation for patients with disabilities. The user is given a report of the patient's information and tasks needed to be done in preparation for their appointment. Once given the report, the user moves objects, adjusts the bed, and prepares the environment to fit the patient's needs.
